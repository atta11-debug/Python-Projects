Fake Headlines Generator
========================

Description:
------------
This Python program generates random, funny, and sometimes absurd fake news headlines 
by combining a random subject, action, and place/thing. 
Each time you run it, you'll get a different headline.

Features:
---------
- Uses `random.choice()` to pick words from predefined lists.
- Infinite headline generation until the user types "no".
- Fun mix of political figures, animals, sports, and everyday life.
- Beginner-friendly Python project to practice randomization and user input.

How to Run:
-----------
1. Make sure you have Python installed (version 3.x recommended).
2. Save the file as `fake_headlines.py`.
3. Open a terminal/command prompt in the same directory.
4. Run:
   python fake_headlines.py

Example Output:
---------------
BREAKING NEWS: A Lahore cat cuddles inside parliament!
Do you want another headline? (yes/no): yes

BREAKING NEWS: Shahid Afridi orders onions from Anarkali Bazaar!
Do you want another headline? (yes/no): no
