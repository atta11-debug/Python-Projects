Python Mini Projects - README
=============================

This folder contains multiple Python mini-projects, each demonstrating core programming concepts such as file handling, object-oriented programming, loops, conditionals, and user interaction.  
All projects are written in Python 3 and run in the terminal/command prompt.

----------------------------------------------------
1. bankaccount.py
----------------------------------------------------
Description:
Simulates a simple bank account system where users can:
- Create an account
- Deposit money
- Withdraw money
- Check balance

Key Concepts:
- Classes & Objects
- Encapsulation
- Basic input validation

How to Run:
python bankaccount.py

----------------------------------------------------
2. employee_payroll.py
----------------------------------------------------
Description:
Manages employee payroll by storing and calculating salary details.  
Allows adding employees, viewing payroll records, and computing total payouts.

Key Concepts:
- Classes & Objects
- Dictionaries and lists
- Simple calculations

How to Run:
python employee_payroll.py

----------------------------------------------------
3. library_with_fileSaving.py
----------------------------------------------------
Description:

Key Concepts:

- Classes & Objects

How to Run:
python library_with_fileSaving.py
