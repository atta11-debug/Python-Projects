Student Details and Average Marks Calculator

Description:
------------
This program stores and displays a student's details including:
  - Name
  - Roll number
  - Subjects and marks
It also calculates and displays the average marks of the student.

Features:
---------
1. Accepts student data (name, roll number, subjects, marks) via variables.
2. Prints the student's name and roll number.
3. Lists each subject with its corresponding mark.
4. Calculates and displays the average marks with two decimal precision.

How It Works:
-------------
- The 'student' class stores the student's data and has:
    1. calculate_average(marks) → Calculates average marks.
    2. show_details() → Displays all student information.

Requirements:
-------------
- Python 3.x

Run Instructions:
-----------------
1. Save the file (e.g., student_details.py).
2. Run it in a terminal or IDE:
      python student_details.py
