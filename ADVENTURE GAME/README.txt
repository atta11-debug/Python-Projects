 DESCRIPTION

This is a simple interactive text adventure game where the player makes choices that affect the outcome of the story.

The game starts with the player on a dirt road and offers different paths to take. 
Depending on the choices made, the player may:
- Find treasure and win üéâ
- Get eaten by an alligator üêä
- Lose the game in various ways

The story has multiple endings based on player decisions.

--------------------
 HOW TO RUN
--------------------
1. Make sure you have Python 3 installed on your system.
2. Open a terminal or command prompt in this project folder.
3. Run:
   python main.py
4. Follow the on-screen prompts and type your answers.

--------------------
 GAMEPLAY NOTES
--------------------
- Type exactly the option shown (e.g., "left", "right", "swim", "walk", "yes", "no").
- Inputs are case-insensitive, so "LEFT" or "left" will both work.
- Invalid input will result in losing the game.
