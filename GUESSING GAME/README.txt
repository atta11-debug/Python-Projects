Password Guessing Game
======================

Description:
------------
This is a simple Python-based password guessing game. 
The user selects a difficulty level (Easy, Medium, Hard), and the program randomly 
chooses a secret word from that level's list. The player guesses until they find the correct password, 
receiving hints about correctly placed letters after each guess.

Features:
---------
- Three difficulty levels with different word lists.
- Hints after each wrong guess, showing correct letters in their exact positions.
- Tracks the number of attempts taken to guess the password.
- Beginner-friendly project for practicing loops, conditionals, and string handling.
