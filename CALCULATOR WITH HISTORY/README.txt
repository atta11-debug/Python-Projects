# Simple Calculator in Python

## Description
This is a simple Python calculator program that can perform basic arithmetic operations:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

It also supports:
- Viewing calculation history
- Clearing calculation history
- Storing results in a history file

The program stores history in a file named `history.txt`.

---
